# GEE33 — Your Edge, Engineered

![GEE33 Logo](assets/favicon.svg)

A modern, business-focused website for GEE33, an AI-powered software development company based in South Africa.

## 🚀 Overview

GEE33 builds custom software and AI systems that give businesses an unfair advantage. This website is designed to speak directly to business decision-makers — focusing on outcomes, not technical jargon.

## 📄 Pages

- **Home** (`index.html`) — Main landing page with hero, differentiators, results, and services overview
- **Why Us** (`why-us.html`) — Detailed breakdown of what makes GEE33 different
- **Results** (`results.html`) — Case studies and success stories with concrete metrics
- **Services** (`services.html`) — Full service offerings with benefits and process
- **About** (`about.html`) — Company story, values, and team philosophy
- **Contact** (`contact.html`) — Contact form and booking information

## 🎨 Design Features

- **Custom cursor** with magnetic hover effects
- **Animated gradient orbs** that respond to scroll and mouse position
- **Scroll-reveal animations** using Intersection Observer
- **Responsive design** for all screen sizes
- **Dark mode** aesthetic with accent colors

## 🛠 Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript** — Vanilla JS for interactions (no frameworks)
- **Fonts** — Cabinet Grotesk & Satoshi via Fontshare

## 🚀 Quick Start

1. Clone the repository
2. Open `index.html` in a browser
3. No build process required — pure HTML/CSS/JS

## 🎯 Key Messaging

The website copy focuses on:

- **Business outcomes** over technical features
- **Speed to market** (4-8 weeks vs 6-12 months)
- **Fixed pricing** with no surprises
- **Direct access** to builders (no middlemen)
- **Concrete metrics** from real case studies
